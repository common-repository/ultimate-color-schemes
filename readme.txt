20+ new color schemes for your admin with ability to assign color schemes to user roles.

== Source Code ==

The uncompiled versions of the JavaScript and CSS files can be found in the following locations within this plugin:
- Admin JavaScript: `assets/admin/admin.js`, `assets/admin/settings-page/scripts.js`
- Admin CSS: `assets/admin/admin.scss`, `assets/admin/settings-page/styles.scss`
- Editor CSS: `assets/editor.scss`
- Schemes CSS: `assets/schemes/`

All non-distributed files are located in the `assets/` directory. The compiled files are output to the `dist/` directory.
